# Simple Chess Engine

## How to Play

1. Clone the repository:
    ```bash
    git clone git@github.com:nolan-reilly/PyChess.git
    ```
2. Move into the current directory/Folder
   ```bash
   cd PyChess
   ```

3. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

4. Start the game:
    ```bash
    python ChessMain.py
    ```
